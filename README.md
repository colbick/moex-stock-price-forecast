# MOEX Stock Price Forecast

Учебный проект по прогнозу цен акций на Московской бирже на 1, 3 или 7 дней вперед (Горизонт инвестирования).

## Что делает проект
- проверяет наличие тикера в базе Московской биржи;
- загружает исторические торговые данные;
- строит график цены закрытия;
- формирует признаки для модели;
- обучает несколько моделей линейной регрессии;
- сравнивает модели по метрикам качества;
- выводит прогнозное значение цены.

## Используемые данные
Источник данных: ISS MOEX

## Используемые библиотеки
- pandas
- requests
- matplotlib
- seaborn
- scikit-learn

## Запуск
Откройте ноутбук `Питон_проект.ipynb` и запускайте ячейки по порядку.

## Примечание
Проект выполнен в учебных целях и не является инвестиционной рекомендацией.
